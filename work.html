Meet App

The objective was to build a serverless, progressive web application with React using a test-driven development technique. The application uses the Google Calendar API and AWS Lambda to fetch upcoming web development events. Users can search for a city and get a list of events hosted in that city. During the testing process I converted the key features of the app into user stories, then used the “Given-When-Then” syntax to define a full list of scenarios. You can view the list here:
