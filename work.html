Meet App

The objective was to build a serverless, progressive web application with React using a test-driven development technique. The application uses the Google Calendar API and AWS Lambda to fetch upcoming web development events. Users can search for a city and get a list of events hosted in that city. In addition, two graphs built with the Recharts library make it easy to see how many events are about to take place in each city and how popular the different kinds of event genres are.

Technologies:

React

Google Calendar API

AWS Lambda
